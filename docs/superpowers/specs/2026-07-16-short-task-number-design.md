# 縮短顯示用任務編號

## 目的

內部任務 ID 為安全檔名格式，不適合人員溝通。Dashboard 與 Matrix 通知改顯示較短、可反查的任務編號，例如：

```text
20260716-114946-q3fnoi
```

## 規則

- 日期與時間使用 Asia/Shanghai（UTC+8）。
- 保留內部 ID 最後的隨機碼，供精確反查。
- 內部 queue ID、檔名、API 查詢及日誌索引不變。
- 無法辨識的舊 ID 原樣顯示。
- Dashboard 與通知共用同一個轉換函式。

## 驗證

- 驗證 `2026-07-16T03-49-46-175Z-____-q3fnoi` 轉為 `20260716-114946-q3fnoi`。
- 驗證 Dashboard 與通知使用縮短編號。
- 執行完整測試與真實 Codex smoke test。
